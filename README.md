# SCT_DS_2
# SkillCraft Internship Task 2
## Objective
Perform Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic Dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Data Cleaning
- Filled missing Age values using median.
- Filled missing Embarked values using mode.
- Removed Cabin column due to excessive missing values.

## Exploratory Data Analysis
- Survival Count Analysis
- Gender Distribution Analysis
- Age Distribution Analysis
- Passenger Class vs Survival Rate

## Conclusion
The analysis shows that passenger class and gender had a strong influence on survival rates. Most passengers belonged to lower age groups, and survival patterns varied significantly across classes.
